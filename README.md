# 🛒 Amazon Product Scraper

## 📌 Problem

* Manually collecting product information from Amazon is slow and repetitive.
* Extracting data from multiple search result pages requires significant time and effort.
* Manual data collection can lead to inconsistencies and human errors.
* Organizing the collected data for analysis is difficult.

## ✅ Solution

* Automated the entire data collection process using Python.
* Used Selenium to search products and navigate through multiple pages.
* Parsed the webpage using BeautifulSoup to extract product information.
* Stored all extracted data in a structured CSV file for easy analysis.

## 📊 Output

* CSV file containing scraped Amazon product data.
* Product Title
* Product Price
* Product Rating
* Organized and structured dataset ready for analysis or business use.

## 🛠 Tools & Technologies

* Python
* Selenium
* BeautifulSoup (bs4)
* Chrome WebDriver
* WebDriver Manager
* CSV Module
